ABOUT

Web制作を通して、伝えたい想いや価値が、
きちんと届くサイトづくりを心がけています。

これまで営業職として約30年間、
お客様の課題や要望を丁寧に聞き取り、
形にする仕事に携わってきました。

その経験を活かし、
Web制作においても一方的な提案ではなく、
目的や背景を大切にしたサイトづくりを心がけています。

SERVICE

Webデザイン

ヒアリングをもとに、サイトの目的やターゲットに合わせたデザインを制作します。見た目の美しさだけではなく、「何を伝えたいか」「どう見せるか」を大切にした分かりやいデザインを心がけています。

コーディング

HTML / CSS / JavaScript を用いて、デザインを忠実に再現し、スマートフォンにも対応したWebサイトを制作します。修正や更新をしやすい、シンプルで丁寧な実装を心がけています。

SKILL

Webサイトの構造作成、レイアウト調整、レスポンシブ対応を行います。また、メニュー開閉やアニメーションなど、サイトに必要な基本的な動きを実装します。

WordPressでは、お知らせや文章をご自身で更新できるため、運用しやすいサイトをご提案します。
